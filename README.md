# Vietnam Informal Economy Analysis

Interactive visualization and analysis of Vietnam's informal employment patterns by province.

Designed and built by Miel Wewerka

## 🌐 Live Site

**https://mielwewerka.github.io/vietnam-informal-economy/**
//

## 🚀 Local Development

```bash
npm install
npm start
```

Opens at `http://localhost:3000`

## 📦 Deploy

```bash
npm run build
npm run deploy
```
